Analysis of Determinants of Poverty in Rwanda

  • Analyzed ten years of EICV data with Stata and Python to examine how education, employment, demographics, and location influence poverty in Rwanda.
  • Applied binary and ordered probit models to distinguish between moderate and severe poverty, providing more nuanced insights compared to basic methods.
  • Compared results across EICV4, EICV5, and EICV7, highlighting persistent structural inequalities despite economic growth.

Tools: Stata, Python, Econometrics, Survey Data

Health Analytics Dashboard (Flask + MySQL)

A Flask + MySQL web application that analyzes outpatient health visits and drug stock data using a relational database and server-side analytics.

This project demonstrates data modeling, SQL analytics, backend development, and data quality management.

Tools: Flask, MySQL, SQL, Python, Web Analytics
